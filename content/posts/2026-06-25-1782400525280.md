Популярное расширение для блокировки рекламы в YouTube под названием Adblock for YouTube было обнаружено с возможностью выполнения вредоносного кода. Исследование, проведённое Island, показало, что расширение может выполнять произвольный JavaScript-код, что делает его потенциально опасным для пользователей.

Adblock for YouTube — это расширение для Google Chrome, которое имеет более 10 миллионов установок и отмечено как «Featured» в Chrome Web Store. Оно позиционируется как инструмент для блокировки рекламы на YouTube и других сайтах. Однако, несмотря на свою популярность, расширение скрывает опасную функцию — возможность выполнения произвольного JavaScript-кода.

Расширение Adblock for YouTube было обнаружено с «спящим» скриптом, который может быть активирован для выполнения вредоносного кода. Это означает, что злоумышленник мог бы использовать расширение для внедрения вредоносного кода в браузер пользователя. Исследователи отметили, что расширение не было удалено из Chrome Web Store, несмотря на обнаруженную уязвимость.

После обнаружения уязвимости разработчики Adblock for YouTube не предоставили официальных комментариев. Однако, Google, компания, которая владеет Chrome Web Store, может предпринять меры по удалению расширения или предупреждению пользователей о потенциальных угрозах.

Для пользователей это означает, что они должны быть осторожны при установке расширений для блокировки рекламы. Особенно важно проверять репутацию расширений и избегать тех, которые могут скрывать вредоносные функции. Также рекомендуется использовать несколько источников для проверки безопасности расширений перед их установкой.
