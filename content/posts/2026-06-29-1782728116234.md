Microsoft удалила 119 вредоносных расширений для браузера Edge, которые скрывали вредоносный код в изображениях и шрифтах. Эти расширения, получившие название StegoAd, использовали метод стеганографии — скрытие данных внутри других файлов — для маскировки вредоносного кода.

StegoAd — это комбинация стеганографии и адверт-софта, и Microsoft связывает 119 расширений с одним угрозообъектом, который действует с 2021 года. Эти расширения были обнаружены в магазине дополнений Edge и использовались для кражи учётных данных пользователей и запуска фрод-рекламы. После установки расширения они активировались через несколько дней, чтобы начать свою вредоносную деятельность.

Microsoft обнаружила StegoAd, когда заметила, что расширения загружают вредоносный код из изображений и шрифтов, которые выглядели как обычные файлы. Эти расширения использовали сложные методы стеганографии, чтобы скрыть вредоносный код внутри изображений и шрифтов, что делало их трудными для обнаружения традиционными методами безопасности. После установки расширения они активировались через несколько дней, чтобы начать кражу данных и запуск фрод-рекламы.

Удаление этих расширений — часть усилий Microsoft по защите пользователей от вредоносного ПО. Microsoft продолжает работать над улучшением безопасности браузеров и магазинов дополнений, чтобы предотвратить подобные угрозы в будущем. Это удаление подчеркивает важность использования проверенных и надежных расширений, чтобы избежать попадания в ловушки вредоносного ПО.

Для пользователей браузера Edge это значит, что они должны быть осторожны при установке расширений и всегда проверять их репутацию перед установкой. Microsoft рекомендует использовать только расширения из официальных магазинов и регулярно обновлять браузер, чтобы защитить себя от новых угроз.
