Киберугроза, связанная с вредоносным ПО Miasma, распространяется всё шире. Исследователи обнаружили новые атаки, направленные на npm-пакеты и GitHub Actions. Это часть более крупной кампании, включающей вредоносные программы Mini Shai-Hulud, Miasma и Hades.

История атаки началась с компрометации npm-пакетов, таких как LeoPlatform и RStreams. Хакеры внедрили вредоносный код в эти пакеты, которые затем распространялись среди разработчиков. Аналогичная тактика была применена к Go-экосистеме, где вредоносное ПО также было внедрено в популярные проекты.

Ключевые детали атаки включают:
- Внедрение вредоносного кода в npm-пакеты LeoPlatform и RStreams.
- Использование GitHub Actions для распространения вредоносного ПО.
- Компрометация Go-экосистемы через внедрение вредоносного кода в популярные проекты.

Рынок реагирует на угрозу, увеличивая мониторинг активности в npm и GitHub Actions. Компании, использующие эти инструменты, рекомендуют проводить регулярные аудиты зависимостей и отключать неиспользуемые GitHub Actions.

Для разработчиков это означает повышенную бдительность при работе с npm-пакетами и GitHub Actions. Необходимо тщательно проверять исходный код перед его использованием и избегать подозрительных зависимостей. Также важно следить за обновлениями безопасности и реагировать на новые угрозы оперативно.
